Kahlan

Kahlan is a full-featured Unit/BDD Test Framework for PHP 5.4+, a la RSpec/JSpec using a Jasmine style notation.

It embrace the KISS principle and makes Unit/BDD testing fun again.

Uniqueness: Kahlan allow stub or monkey patch your code directly like in Ruby or JavaScript without any required PECL-extentions.

Requirements

  • PHP 5.4+
  • Using an autoloader in your project (Composer for example)
  • Xdebug if you want to perform code coverage analysis.

Installation

git clone git@github.com:crysalead/kahlan.git
cd kahlan
composer install
bin/kahlan              # to run specs or,
bin/kahlan --coverage=4 # to run specs with coverage info for namespaces, classes & methods (require xdebug)
